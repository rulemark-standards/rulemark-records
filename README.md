# RuleMark Records

Public artifact store for RuleMark frozen standards. This repository stores
normative files; it does not define or revise their content.

## Publishing rule

Before adding or replacing an artifact, read the canonical
[Publishing Integrity Policy](https://github.com/rulemark-standards/rulemark-canonical-archive/blob/main/docs/PUBLISHING-INTEGRITY-POLICY.md).

Every release must pass the RuleMark Integrity Gate. Do not edit, regenerate,
rename, or silently replace a frozen PDF. A content change requires a new
version. A mapping correction must restore the previously frozen and signed
bytes.
